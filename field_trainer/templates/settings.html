Touch Sensor Calibration
How to Use

Three tools are available for touch sensor setup and validation:

Live Reading
  • Purpose: Monitor real-time sensor data to verify hardware is working
  • Duration: 10 seconds of live readings at 5Hz (5 readings per second)
  • How to read:
    • Mag: Current sensor magnitude (deviation from rest)
    • Threshold: Minimum magnitude required to trigger touch
    • ðŸŽ¯ TOUCH! appears when magnitude exceeds threshold
  • When to use: Before calibration to confirm sensor responds to touch/movement
Test Mode
  • Purpose: Validate threshold setting by counting actual touches
  • Duration: 10 seconds with touch counting enabled
  • How to use:
    • Tap the device 5 times during the test period
    • Each tap should produce a beep and increment touch count
    • Final report shows total touches detected
  • Expected result: 5 taps = 5 beeps = 5 touches detected
  • When to use: After setting threshold to verify it works correctly
Calibrate
  • Purpose: Automatically determine optimal threshold for the device
  • Duration: 30 seconds total (~6 seconds per tap)
  • How it works:
    • Wizard prompts you to tap device 5 times
    • Analyzes tap magnitudes to calculate optimal threshold
    • Auto-adjusts sensitivity if no taps detected
  • When to use: First time setup or when recalibrating a device
Manual Threshold Adjustment
  • Click threshold value in table to edit manually
  • Typical range: 0.15g - 0.30g
  • Lower values = more sensitive (may trigger false positives)
  • Higher values = less sensitive (may miss light touches)
